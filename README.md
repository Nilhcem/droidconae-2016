# Droidcon Tunisia 2016 Conference App

Sample app for the "2016 Android Developer Toolbox" talk at droidcon tunisia

## Run Mock Server

```bash
cd mockserver
npm install
node server
```

You can then access it through:

```
http://localhost:8990/
```
